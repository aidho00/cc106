02 Design

UI/UX Review

CARP Principles

  • Contrast — make different things VERY different
  • Alignment — everything visually connected
  • Repetition — consistent patterns
  • Proximity — related items grouped

Key Rules

  • 60-30-10 color rule
  • Max 2 fonts
  • 3-6 links in primary navigation
  • Core task in ≤ 3 clear steps
